# Wire Motion Flipbook — Next.js

快速说明：在项目根目录运行以下命令来安装并启动开发服务器（需要 Node.js 已安装）。

Windows (PowerShell) 注意事项：如果 `npm` 被 PowerShell 的执行策略阻止，可以用 `npm.cmd`：

```powershell
npm.cmd install
npm.cmd run dev
```

或在普通命令提示符 (cmd.exe) 中运行：

```cmd
npm install
npm run dev
```

浏览器打开: http://localhost:3000

上传 `processed_intermediate_data.xlsx`，在左侧选择 "Upload Excel"。使用 ← → 翻页。
